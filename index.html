DIPWallet

DIPWallet is an easy to use e-wallet that stores all your loyalty and membership cards. Just use your phone to scan the QR code given to each card at every retail stores and you're good to go. DIPWallet has been developed for the purpose of digitizing the Indonesian commerce by a small step. It is developed only on Android and took about three months to complete. The front-end was designed with Google Material Design and the database is built with SQLiteDatabase.

  • Languages: Java, XML
  • URL: Bitbucket Repository
  • Landing Page: Landing Page

Fabflix Web Application and Android App

Fabflix is a web-application that is built to resembles an e-commerce site selling movies online. This was built as part of a UCI Project in Database and Web Apps course. The web-application had functionalities such as caching, connection pooling, advanced search and also auto-fill in search for enhanced user experience. This was my favorite project out of all because it really developed my understanding of how REST architecture works since I also built a small Android app that does a simple GET request through the Fabflix web-service.

  • Languages: HTML, CSS, Javascript, Java, XML, MySQL
  • URL: Github Repository

Zotfeed - Android Application

I am actively working on this project right now. This is also an Android application that is used to centralize all of UC Irvine's media channels: KUCI (UCI's Radio), New University (UCI's newspaper), AnteaterTV (UCI's campus TV station). I have finished implementing the KUCI feature of this application and so users can stream the radio from the app. I have also built a web-service with Python so users can retrieve schedules of radio shows in real timeThe next steps is to use YoutubeAPI for AnteaterTV and of course RSSFeeds for getting all the newspaper's articles.

  • Languages: Java, XML, Python, MySQL
